# Mandelbrot Viewer

##Project Overview
This project implements a Mandelbrot set viewer using **SFML 2.4.0**.  
It was developed for the lab assignment and includes extra credit features.

The code is located in the branch **`feature/my-change`**.  
Please switch to that branch to view, build, and run the project.
